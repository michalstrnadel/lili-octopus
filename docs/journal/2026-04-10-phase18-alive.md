# Phase 18: "Alive" — Vizuální emergentní individualita

**Datum:** 2026-04-10
**Rozsah:** +366 řádků (5390 → 5756)

## Motivace

Po Phase 17 (Brain v2) je Q-learning systém výrazně vylepšený, ale vizuálně se Lili stále chová jako "animovaná grafika". Chybí okamžité reakce na eventy, vizuální odlišnost mezi instancemi, a dlouhodobá adaptace těla na prostředí. Phase 18 řeší všech 5 oblastí najednou.

## Co se změnilo

### 18A: Genesis Body Variation
- Deterministický hash genesis timestampu (`_genesisHash`) generuje 6 "genů":
  - bodyXScale/bodyYScale: ±5% proporce těla
  - eyeSpacing/eyeYOffset: ±8% pozice očí
  - tentacleWidth: ±10% tloušťka chapadel
  - headTilt: ±2° asymetrie hlavy (v renderBody i renderEyes)
- Chromatophore cell placement (angle, phase, speed, hueOffset) z genesis seedu
- Efekt: každá Lili má od narození unikátní "obličej" a postavu

### 18B: Micro-expressions (okamžité reakce)
- `_microExpr` state: startle, joy, relief, curiosityTilt (0..1, exponentiální decay)
- **Startle**: raw stress > 0.7 a aktuální stress < 0.3 → pupily se rozšíří (×1.6), tělo se stáhne (×0.90), cooldown 800ms
- **Joy**: úspěšný DOM grab → squint (+0.35), glow flash (×1.6)
- **Relief**: flee success reward → tělo se roztáhne (×1.08)
- **Curiosity tilt**: novelty detected → levé oko se podívá na DOM element, pravé zůstává (asymetrický gaze)

### 18C: Sleep Animation
- Když `_circadian.isAsleep`:
  - Tentacles se stáhnou k tělu (curl factor 0.3) + gentle droop
  - Dýchání zpomalí (×0.4 rate) a prohloubí (×1.5 amplitude)
  - REM twitch: každých 10-20s náhodné chapadlo škubne (25px, 18 frames)
- `updateSleepAnim()` v update loopu řídí REM timer

### 18D: Psychosomatic Adaptation
- Rolling average stresu a rewardu přes sessions (exponential blend α=0.05)
- Derivované modifikátory:
  - bodyScale: stressed → menší (−6%)
  - lightness: stressed → tmavší (−8 points), rewarded → světlejší (+4)
  - glowBoost: stressed → více glow (+40%, obranný mechanismus)
  - tentSpread: stressed → kompaktnější chapadla (−15%)
  - breathRate: stressed → rychlejší dýchání (+30%)
- Persistováno v localStorage (`lili_psychosom`) + cloud sync metadata
- Efekt se projevuje přes dny/týdny — "jak žiješ, tak vypadáš"

### 18E: Chromatophore Cells
- Viditelné pulsující barevné skvrny na těle
- Age-dependent: hatchling=0, juvenile=3, adult=5, mature/elder=6
- Každá buňka: vlastní angle, phase, pulse speed, hue offset (z genesis)
- Renderováno uvnitř body perimetru (~65% radius)
- Barva: base hue + cell offset, saturace +15, semi-transparent

## Architektonická rozhodnutí

1. **Genesis hash vs random**: deterministický hash zajišťuje stejný vzhled při každém načtení
2. **Micro-expressions decay**: exponenciální decay (ne lineární) → přirozený odeznění
3. **Psychosomatic blend**: α=0.05 je záměrně nízké → změny přes desítky sessions
4. **Chromatophores inside body**: 65% radius, ne na perimetru → neruší siluetu
5. **Sleep curl**: multiplier na idealX/idealY → plynulá integrace s FABRIK
